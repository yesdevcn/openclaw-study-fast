## 🧠 一句话理解 OpenClaw

> 如果说 ChatGPT 是「一个很聪明的人」，那 OpenClaw 更像是「一个会用无数工具、会跑自动化脚本、还能 24 小时值班的超级助理」。

简单讲：

- **底层是各种大模型**（GPT、Claude、国产 DeepSeek/Kimi 等）
- **上面包了一层「大脑 + 工具系统」**，能：  
  - 读写你本地/云端文件  
  - 调用各种 Skills（比如：查天气、拉日报、跑脚本）  
  - 通过飞书/企微/QQ 等 IM 和你对话  
- 你要做的，就是：  
  - 把它**装起来**  
  - 告诉它你有哪些工具、想让它帮你干什么  

详细介绍请看：`docs/01-what-is-openclaw.md`。
