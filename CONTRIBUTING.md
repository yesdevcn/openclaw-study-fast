# 贡献指南

感谢您对本项目的关注！为了保证项目的质量和可持续发展，我们鼓励社区参与贡献。

## 如何贡献

### 提交 Issue
如果您在使用过程中发现问题或有建议，请提交 [GitHub Issue](https://github.com/openclaw/openclaw/issues)。

### 提交代码
1. Fork 本仓库。
2. 创建新分支：`git checkout -b feature-branch-name`
3. 提交修改：`git commit -m 'Add new feature'`
4. 推送到分支：`git push origin feature-branch-name`
5. 提交 Pull Request。

### 贡献内容
- 修复 Bug
- 提交功能改进
- 优化文档（包括翻译与排版）

---

感谢您的支持与贡献！
